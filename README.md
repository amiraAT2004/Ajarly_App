# Ajarly_App
 مشروع "أجري" تطبيق لتأجير العقارات في ليبيا، تم بناؤه باستخدام Flutter وFirebase لتبسيط عملية البحث عن عقار مناسب. يعرض التطبيق العقارات بطريقة منظمة مع تفاصيل كاملة (صور، سعر، مواصفات) ويمكن الملاك من إضافة عقاراتهم بسهولة. نُفذ المشروع كعمل أكاديمي من قبل طالبات كلية التقنية الإلكترونية بطرابلس لتعزيز التحول الرقمي في القطاع السكني.
